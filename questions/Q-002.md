# Progress saving

**Area** Save system

**Observation:**

After the game is force closed, some gameplay progress is not saved (e. g. dungeon run progress). It is unclear which game states are intended to be saved.

**Questions:**

1. Is progress during an active dungeon run expected to be saved?
2. If a player unlocks temporary access to the second character by watching an advertisement but closes the game before starting or completing the run, should the temporary access remain available?
