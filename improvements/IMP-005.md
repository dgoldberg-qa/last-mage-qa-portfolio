# Skill Tree entry point is visually misleading

**Area:** Skill tree

**Observation:**
The wisdom level indicator attracts more attention than the character despite being non-interactive. Because it is positioned close to the settings button, users may accidentally open settings instead of the skill tree.

**Suggestion:**
Increase the visual emphasis of the character or reduce the prominence of the wisdom indicator.

**Benefit:**
Improves navigation accuracy and reduces accidental taps.
